    JavaScript is a cross-platform, object- oriented scrpting language. JavaScript is extremely popular for a variety of reasons. It is a small and lightweight language allowing maximum flexibility for developers to take it in a bunch of different directions. JavaScript lives inside a host enviroment (a web browser or Node server), it can be connected to the objects of theseenviroments to provide programmatic control over them.
